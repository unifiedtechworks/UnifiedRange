# Task 005: Public Discovery

## Goal

Build the public setup discovery MVP using mock public passport data.

## Requirements

1. Discover page with public passport cards.
2. Filters for equipment type, category/caliber, manufacturer, model, and use case.
3. Public passport detail page.
4. Reactions UI.
5. Comments placeholder.
6. Report button.

## Privacy Requirements

Public passport should not show serial numbers, exact locations, private notes, purchase records, or image metadata.

## Acceptance Criteria

- Discovery works with mock data.
- Public profiles look sanitized.
- Report button exists.

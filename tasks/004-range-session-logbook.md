# Task 004: Range Session Logbook

## Goal

Build the range-session logging experience.

## Requirements

1. Range-session list page.
2. Range-session detail page.
3. Create/edit form placeholder.
4. Link sessions to equipment passports and projectile profiles.
5. Include target photo placeholder.
6. Include manual group-size or score fields.
7. Include cold-bore/first-shot marker.

## Important Boundary

Do not calculate or display aiming corrections.

## Acceptance Criteria

- User can view mock sessions.
- Session detail shows setup, distance, position, support, notes, and target photo placeholder.

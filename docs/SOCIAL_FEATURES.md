# Social Features

## Social Product Direction

UnifiedRange should not be a generic social media feed.

It should be a structured setup discovery network where users can learn from real-world equipment combinations, range reports, and readiness workflows.

## Public Content Types

- Public equipment passport
- Public range-session summary
- Public target photo
- Public setup review
- Public hunting readiness template
- Public gear checklist
- Public maintenance summary

## Followable Entities

Users can follow:

- Other users
- Equipment types
- Calibers/categories
- Rifle/pistol/bow models
- Optic/sight brands
- Ammo/projectile brands
- Use case tags
- Hunting categories
- Target disciplines

## Discovery Filters

- Equipment type
- Manufacturer
- Model
- Caliber/category
- Optic/sight
- Ammo/projectile
- Use case
- Hunting vs target
- Budget
- Weight class
- Beginner-friendly
- Public target photos available
- Public range reports available

## Reactions

Use practical reactions instead of generic likes:

- Helpful setup
- Similar to mine
- Good hunting build
- Budget-friendly
- Lightweight
- Well documented
- Beginner-friendly

## Comments

Comments should be allowed on public passports and public range reports.

Comments must be reportable.

Future moderation may include automated filters and admin review queues.

## Privacy Requirements

Public sharing should be opt-in.

Public passport previews should clearly show what will and will not be shared.

The app should strip image metadata before public display.

Users should be able to unpublish public content.

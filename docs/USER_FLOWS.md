# User Flows

## New User Onboarding

1. User signs up.
2. User chooses display name and privacy defaults.
3. User sees explanation: private by default, public sharing is optional.
4. User creates first equipment passport.
5. User optionally adds projectile/ammo and optic/sight profiles.
6. User logs first range session.

## Create Equipment Passport

1. User opens Equipment Passports.
2. User selects equipment type: rifle, pistol, bow, crossbow, other.
3. User enters basic setup details.
4. User adds optic/sight and projectile/ammo info.
5. User adds private notes.
6. User saves.
7. Passport remains private by default.

## Log Range Session

1. User opens Range Sessions.
2. User selects equipment passport.
3. User selects projectile/ammo.
4. User enters date, distance, position, support, conditions, notes.
5. User uploads target photo.
6. User enters group size or score manually.
7. User marks cold-bore/first-shot if applicable.
8. User saves session.

## Hunting Readiness

1. User creates hunting checklist.
2. User selects equipment passport.
3. User adds species/season/use case.
4. User checks off readiness tasks.
5. User links recent range sessions.
6. User keeps checklist private.

## Publish Public Passport

1. User opens private passport.
2. User chooses “Create Public Version.”
3. App previews sanitized public data.
4. App warns that serial numbers, exact location, private notes, and EXIF GPS data are excluded.
5. User edits public notes and photos.
6. User publishes.
7. Public passport becomes discoverable.

## Discover Setups

1. User opens Discover.
2. User filters by equipment type, caliber/category, model, optic/sight, projectile/ammo, or use case.
3. User views public passport cards.
4. User opens public passport detail.
5. User reacts, comments, follows user, or follows tags.
